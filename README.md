# Task4

Нужно реализовать несколько команд Unix. Команды, обязательные для
реализации:

### Обязательные:
- echo - вывод текста с обработкой специальных символов
- pwd - вывод текущей рабочей директории
- ls - вывод содержимого директории
### I
- ln - создание ссылок на файлы
### II
- wc - подсчет строк, слов и символов
### III
- cat - вывод содержимого файлов

--------------------------------------------------------------
## echo 
Печатает свои аргументы через пробел, возможны флаги 
### Флаги
    -n - не выводить перевод строки в конце
    -e - включить интерпретацию escape-последовательностей
    -E - отключить интерпретацию escape-последовательностей

Поддерживаемые escape-последовательности

    \n - перевод строки
    \t - табуляция
    \c - остановка вывода
    \\ - обратный слеш

--------------------------------------------------------------
## pwd
Печатает имя текущего каталога
Динамически выделяет буфер под путь. Использует getcwd() для получения текущего пути. Автоматически определяет максимальный размер буфера через pathconf().

--------------------------------------------------------------
## ls
Выводит содержимое директории с поддержкой различных форматов вывода.

### Флаги
    -R - рекурсивный обход поддиректорий
    -l - подробный вывод с атрибутами файлов
    -g - вывод имени группы владельца

### Реализованные функции

    get_owner_name() - получение имени владельца по UID
    get_group_name() - получение имени группы по GID
    get_permissions() - форматирование прав доступа
    print_file_info() - вывод информации о файле
    list_directory() - рекурсивный обход директории

### Особенности

    Подробный вывод с типами файлов, правами, владельцем, размером и временем
    Обработка символических ссылок с выводом цели
    Сортировка файлов по алфавиту
    Рекурсивный обход с форматированием вывода
    
--------------------------------------------------------------
# ln
Создает жесткие или символические ссылки на файлы.

    без флага - жесткая ссылка
    -s        - символическая ссылка

Реализованные функции

    Создание жестких ссылок через link()
    Создание символических ссылок через symlink()

--------------------------------------------------------------
# wc filename
Результат: filename кол-во_строк кол-во_слов кол-во_символов
Возможен список имен файлов; в этом случае подобная информация выдаётся о каждом
файле.
Реализованные функции

    count_file_stats() - подсчет статистики для одного файла
    print_stats() - вывод статистики для файла
    print_total_stats() - вывод общей статистики
Сначала подводим статистику одного файла, потом общую

--------------------------------------------------------------
# cat filenames
Выводит содержимое одного или нескольких файлов.
Возможен флаг:
-n – с нумерацией строк (если файлов несколько, то нумерация сквозная)

### Флаги

    -n - нумерация всех строк вывода


Реализованные функции

    Динамическое управление списком файлов
    Нумерация строк с выравниванием
    Последовательная обработка нескольких файлов
