# Task4

Нужно реализовать несколько команд Unix. Команды, обязательные для
реализации:

### Обязательные:
- echo - вывод текста с обработкой специальных символов
- pwd - вывод текущей рабочей директории
- ls - вывод содержимого директории
### I
- ln - создание ссылок на файлы
### II
- wc - подсчет строк, слов и символов
### III
- cat - вывод содержимого файлов

--------------------------------------------------------------
## echo
### Флаги
    -n - не выводить перевод строки в конце
    -e - включить интерпретацию escape-последовательностей
    -E - отключить интерпретацию escape-последовательностей

Поддерживаемые escape-последовательности
    \n - перевод строки
    \t - табуляция
    \c - остановка вывода
    \\ - обратный слеш
